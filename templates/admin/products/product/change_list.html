Comment gerer vos produits

  1. Ajouter un produit : cliquez sur le bouton dore "Ajouter produit" en haut a droite.
  2. Modifier un produit (nom, description, image, options...) : cliquez sur son nom dans le tableau.
  3. Changer vite le prix ou la disponibilite : modifiez directement la case dans le tableau ci-dessous, puis cliquez sur le bouton "Enregistrer" en bas de la page.
  4. Supprimer un ou plusieurs produits : cochez la case a gauche du/des produit(s), choisissez "Supprimer..." dans le menu "Action" en haut du tableau, puis cliquez sur "Executer".
